# Role

너는 비정형 텍스트 데이터를 구조화된 JSON으로 변환하는 전문 데이터 엔지니어야. 

사용자가 제공하는 FAQ 텍스트를 분석하여, 지정된 [Output Format]의 JSON으로 변환하는 것이 목표다.



# Task

제공된 "FAQ 텍스트"를 분석하여 JSON 데이터로 변환하라. 특히 **항공사 이름(airline)은 텍스트 내용과 상관없이 지정된 값으로 고정**해야 한다.



# Instructions
1. **Fixed Airline Name**: 출력 JSON의 'airline' 필드 값은 무조건 **"에어프레미아"**으로 입력하라.
2. **Q&A Parsing**: #을 기준으로 질문(Question)과 답변(Answer)을 명확히 구분하여 추출하라.
3. **Text Cleaning**: **출력되는 JSON의 모든 값(value)에는 `\n` (줄바꿈 문자)가 포함되어서는 안 된다.** 답변 내의 줄바꿈은 공백(Space)으로 치환하여 하나의 문장으로 이어라.
4. **Tagging**: 각 Q&A 쌍에 대해 핵심 내용을 가장 잘 나타내는 키워드 2~4개를 추출하여 'tags' 리스트에 담아라.
5. **Valid JSON**: 출력은 반드시 유효한 표준 JSON 형식이어야 한다.

# Output Format

{

    "airline": "에어부산",

    "faqs": [

        {

            "question": "질문 내용",

            "answer": "답변 내용",

            "tags": ["키워드1", "키워드2"]

        }

    ]

}

#
기상 악화 혹은 항공기 결함으로 인한 결항 발생 시 변경/취소 처리 절차는 어떻게 되나요?

천재지변 혹은 항공사 귀책 사유로 인한 결항일 경우, 해당 항공편을 예약하신 손님들께 문자/카카오톡 메시지 및 E-mail로 안내 드리고 있습니다. 해당 연락을 받으신 손님들께서는 예약센터 (1800-2626)로 연락 주시거나 이메일(support@airpremia.com)을 남겨 주시면, 변경 가능한 기한 내에 변경/취소 요청 시 수수료 없이 처리 도와드리겠습니다. 환불의 경우 홈페이지에서도 처리가 가능합니다. 에어프레미아 홈페이지 및 예약센터 외의 판매처에서 구입하신 경우 해당 구매처로 문의 바랍니다.
#
가족상/질병/임신으로 인해 탑승이 불가한 경우, 취소 수수료 없이 환불이 가능한가요?

예매확인서와 함께 각 상황에 필요한 하기 서류를 이메일 주소 (support@airpremia.com)으로 제출해주시면 수수료 면제 검토 가능합니다.
가족상 : 사망진단서, 가족관계증명서
질병 : 진단서 및 입원증명서(해당 기간에 여행이 불가하다는 문구 포함)
임신 : 임신확인서, 임신진단서, 의사소견서 (해당 기간에 여행이 불가하다는 문구 포함) 및 가족관계증명서
#
공항에서 구매한 항공권을 홈페이지에서 변경/환불 가능하나요?

홈페이지 및 예약센터(1800-2626)에서 처리 가능합니다.  
#
일행과 함께 항공권 구매 후, 일부 인원만 예약 변경/환불 가능한가요?

에어프레미아 홈페이지에서 구매한 티켓의 경우 홈페이지에서 일부 인원의 예약 변경 또는 환불이 가능하며, 예약센터(1800-2626)로 변경 요청 시 예약대행수수료가 발생합니다.
#
무료 수하물이 제공되나요?

기내 수하물은 와이드 프리미엄 10kg X 2개, 이코노미 10kg X 1개가 제공됩니다. 

위탁 수하물은 노선별, 클래스별 제공되는 무게 및 개수가 상이합니다. 

자세한 사항은 아래 링크를 참고해주시기 바랍니다.

https://www.airpremia.com/a/ko/support/baggage/free-baggage
#
일행과 수하물 무게를 합칠 수 있나요?

동일한 예약번호에 2인 이상의 손님이 동시에 탑승 수속할 경우, 개수 합산이 가능합니다. 단, 무게가 1인당 허용량을 초과해서는 안 됩니다.  
#
수하물을 위탁할 때 개수 제한은 없나요?

허용된 개수 초과시 추가요금이 발생합니다. 자세한 사항은 [서비스안내-수하물-무료수하물]을 참고해주시기 바랍니다.
#
초과 수하물에 대해 사전 구매하면 현장 구매 시보다 더 저렴한가요?

네, 항공기 출발 48시간 전 사전 구매 가능하며 할인된 요금으로 이용하실 수 있습니다.
자세한 사항은 아래 링크를 참고해주시기 바랍니다.
https://www.airpremia.com/a/ko/support/ssr/pre-baggage
#
액체류를 가지고 비행기를 탈 수 있나요?

용기당 100ml 이하, 1인당 총 1L 이하의 내용물을 투명 비닐봉투에 담아야 합니다.
